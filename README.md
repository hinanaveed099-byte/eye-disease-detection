\# Retina OCT Disease Detection using Deep Learning



\## Project Overview



This project is based on Deep Learning for Retina OCT Disease Detection.



The AI model can classify retina OCT images into:



\* NORMAL Retina

\* CNV Disease



The model is trained using EfficientNetB0 and TensorFlow.



\---



\## Features



\* Retina OCT Image Classification
